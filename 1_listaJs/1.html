O que é JavaScript?

É uma linguagem de programação, interpretada, para desenvolvimento web que é executada no navegador do cliente em um sandbox, que limita o acesso aos recursos do sistema operacional, fornecendo interfaces predefinidas, a forma de acesso a recursos do sistema se faz por meio de objetos globais disponibilizados ao navegador, como: document, console, history, local storage. A motivação para o surgimento do JavaScript foi a pouca interatividade das páginas web nos anos 90. enquanto a execução no navegador acontece no computador do usuário para interações rápidas, a execução no servidor ocorre em um servidor remoto para processar dados e interagir com bancos para enviar o resultado ao navegador. A tipagem é dinâmica e forte, permitindo que variáveis possam ser atribuidas e reatribuidas a qualquer momento, sem a necessidade de declarar seu tipo previamente. Os tipos de dados nativos da linguagem são: Number, String, Boolean, Object, Array e Function.
